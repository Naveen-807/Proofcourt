# Sponsor Integrations

ProofCourt keeps the sponsor scope tight: Gensyn AXL, KeeperHub, and 0G.

## Gensyn AXL

Target architecture is a five-node private mesh: requester, worker, verifier-1, verifier-2, and verifier-3. The app should expose peer IDs, message IDs, payload hashes, and transcript hashes.

## KeeperHub

KeeperHub is the execution and settlement automation rail. The target model uses workflow webhook keys for execution and organization keys only for workflow and execution inspection.

## 0G

0G stores the case file and evidence bundle, records verdict metadata, and anchors replayable roots that the UI can inspect.
